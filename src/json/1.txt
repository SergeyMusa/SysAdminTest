 Что такое интерфейс SAS?

 Протокол SCSI через IP, связывающий сервер с хранилищем данных.
 Интерфейс, разработанный для обмена данными с такими устройствами, как жёсткие диски, накопители на оптическом диске , использующий последовательный интерфейс для работы с непосредственно подключаемыми накопителями.
 Интерфейс, разработанный для объединения на одной шине различных по своему назначению устройств, таких как жёсткие диски, накопители на магнитооптических дисках, приводы CD, DVD, стримеры, сканеры, принтеры и т. д.
 Параллельный интерфейс подключения накопителей (жёстких дисков и оптических приводов) к компьютеру.

 Как войти в режим одиночного пользователя (single user mode) в RHEL 7?

 При загрузке системы нужно войти в меню загрузчика GRUB2, нажать 'с', найти строку начинающуюся с 'linux32/vmlinz' и заменить параметр 'ro' на 'rw init=/sysroot/bin/bash'. Нажать ctrl-r для продолжения загрузки.

 Что такое DHCP — назначение?

 Cтандартный протокол, предназначенный для передачи файлов по TCP-сетям (например, Интернет).
 Сетевой протокол, позволяющий компьютерам автоматически получать IP-адрес и другие параметры, необходимые для работы в сети TCP/IP.
 Протокол прикладного уровня для доступа к электронной почте.
 Сетевой протокол прикладного уровня, позволяющий производить удалённое управление операционной системой и туннелирование TCP-соединений (например, для передачи файлов).

 VMware Fusion это…

 Бесплатный продукт, являющийся довольно мощной платформой виртуализации, которая может быть запущена на серверах под управлением хостовых операционных систем Windows и Linux.
 Продукт для создания защищенных политиками безопасности виртуальных машин, которые затем можно распространять по модели SaaS (Software-as-a-Service).
 Бесплатный "проигрыватель" виртуальных машин на основе виртуальной машины VMware Workstation, предназначенный для запуска уже готовых образов виртуальных машин, созданных в других продуктах VMware, а также в Microsoft VirtualPC и Symantec LiveState Recovery.
 Платформа, ориентированная на desktop-пользователей и предназначенная для использования разработчиками ПО, а также профессионалами в сфере ИТ.
 Настольный продукт для виртуализации на платформе Mac от компании Apple.
